# zsh

Install the Zsh configuration with:

- `make -C zsh install`

The `Makefile` targets symlink the configuration files into the home directory.

Important files:
- `alias.zsh`
- `bundle.zsh`
- `export.zsh`
- `Makefile`
- `omz-plugins.zsh`
- `zprofile`
- `zshenv`
- `zshrc`

Use local overrides like `~/.zshrc.local` when needed.
